# Tic Tac Toe
A simple Two Player Tic Tac Toe game.
It's a simple frontend Tic Tac Toe game with no implementation of One vs CPU algorithms
<br/>
<br/>


### Thing Implemented
* Single and Dual Player Choice
* Winning algorithm(How the player win this game)
* Draw Algoritm
